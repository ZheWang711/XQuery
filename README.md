## Setting up CSE232B shell environment
* Change your current directory to project root directory `cd <something depends on your file system>/XQuery`
* Add execution permission (one time): `chmod +x ./antlr4.sh`
* Setting up environment (each time you open a new session): `source ./antlr4.sh`
* After that, you can use `antlr4` and `grun` as you wish
